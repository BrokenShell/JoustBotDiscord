## Highly Random Starting Character

```
Broken:
/make_character Fred the Drunk

DM:

Name: Fred the Drunk
Race: Human
Profession: Rat Catcher
Aspects:
    Mind: 13
    Body: 20
    Spirit: 8
Skills: None
Inventory: Empty

Broken:
/roster

DM:
Broken's Roster:
- Fred the Drunk

```
